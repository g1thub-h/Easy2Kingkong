# Easy2Kingkong
## 简介
本脚本由付导的录播组制作  当前版本为 V1.7 （20200220）
- 1.加入强制获取管理员权限命令，减少误操作（若有问题请及时联系我）

- 2.优化连接速度

- 3.修复推荐、追踪等显示

- 4.修复登陆二维码的出现

- 5.加快[未污染地址](https://www.kingkongapp.com/)连接速度（建议也运行脚本来加快连接速度和其他功能）
